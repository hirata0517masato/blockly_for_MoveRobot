///////////////////////////////////////
使い方(実行形式にて)
Unityにプロジェクト(MoveRobot)をインポートし、ビルドを行う

MoveRobot.exeと同じパスのReadPath.txt
に記述されたファイルを読み込み、
そこに記述されたソースコードを入力とする

MoveRobot.exeと同じパスのReadPathOB.txt
に記述されたファイルを読み込み、
障害物(BOX),旗（Flag)を設置
または、ロボットの初期位置、角度を変更

指定方法は
flag,10,3
box,-4,6
robot,2,5,90

など

///////////////////////////////////////
使い方(webブラウザにて)
リポジトリのMoveRobot_WebGLをWebサーバーに置き、MoveRobot_WebGL/index.htmlにアクセス
テキストボックス（id = 'code'）に記述されたソースコードを入力とする
テキストボックス（id = 'obstacle'）に記述された数値（座標）に障害物(BOX),旗（Flag)を設置,または、ロボットの初期位置、角度を変更

指定方法は
flag,10,3
box,-4,6
robot,2,5,90

など

///////////////////////////////////////
主な対応機能
変数：int,double
条件分岐:if
ループ：for,while
コメント：/* */
時間：delay
   ミリ秒単位で指定した時間次の命令を実行しない
　　delay(1000)

ロボット操作：
  モータ：関数に左右の出力を与える（-100から100）
　　　motor(100,100)

センサー各種：
　タッチセンサー：
　　　bumper()
　　　　　正面がぶつかっているとき　１
	ぶつかっていないとき　０　　　を返す

  距離センサー：
     distance()
	正面の距離を返す
　	引数は0:前 1:後ろ 2:左 3:右 方向を表す

　コンパス：
     compass()
	ロボットの向きを返す
　　　　　時計回りに0~360の値を返す

四則演算や条件式,論理否定に対応
+=や++に対応（ただし、後置、前置に意味はない）
